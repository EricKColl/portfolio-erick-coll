Módulo 01 · Evolución del proyecto

De frontend local a aplicación FullStack real

El Producto 4 conecta el trabajo realizado en los productos anteriores y transforma JobConnect en una solución integrada, con comunicación real entre interfaz, backend, API GraphQL y base de datos.

Producto 02

Frontend preparado

Interfaz con HTML5, CSS3, JavaScript y Bootstrap: pantallas de login, dashboard, usuarios y ofertas/demandas. La persistencia se apoyaba en APIs del navegador.

Producto 03

Backend GraphQL

Servidor Node.js y Express con API GraphQL, conexión a MongoDB, resolvers y operaciones sobre usuarios, publicaciones y elementos seleccionados.

Producto 04

Integración completa

Unión de frontend y API backend mediante Fetch, migración a Mongoose, control de roles, JWT, Socket.io y dashboard actualizado en tiempo real.

01

Integración del frontend con el backend

Las pantallas del frontend dejan de depender de persistencia local y pasan a consultar datos reales mediante el backend GraphQL.

// Completado
02

Mongoose como capa ODM

MongoDB se estructura mediante modelos, esquemas, validaciones, índices y normalización para usuarios, publicaciones y seleccionadas.

// Completado
03

Sesión, JWT y roles

El sistema diferencia administrador, empresa y candidato. Las operaciones protegidas se validan en servidor, no sólo en el cliente.

// Completado
04

Comunicación en tiempo real

Socket.io emite eventos cuando cambian publicaciones, seleccionadas o datos del dashboard, reforzando una experiencia dinámica y moderna.

// Completado

Módulo 04 · Roles y permisos

Experiencia adaptada y control desde servidor

La interfaz se adapta al usuario activo, pero la seguridad importante se valida en el backend mediante JWT, contexto GraphQL y funciones de autorización.
