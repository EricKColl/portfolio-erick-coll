Introducción al Proyecto

El Producto 1 de la asignatura FP.448 — Desarrollo de Aplicaciones Web tiene como objetivo construir, documentar y desplegar un entorno web profesional completo. El equipo BackLord ha implementado el ciclo completo desde la infraestructura local en Docker hasta la migración y verificación del servicio en el servidor remoto de la UOC.

Este proyecto construye un entorno web profesional completo usando contenedores Docker orquestados con Docker Compose. El stack elegido cubre la totalidad del ciclo de despliegue moderno: servidor web, motor de base de datos, herramienta de administración, framework MVC y CMS.

Se ha implementado PHP 8.4 + Apache 2.4 como núcleo de ejecución, MySQL 8.0 como motor de base de datos relacional y phpMyAdmin para la administración visual de esquemas y datos.

Sobre ese cimiento se despliegan Laravel 11.48 — con Composer, migraciones y configuración de entorno vía .env — y WordPress 6.9.1 con base de datos propia, panel de administración verificado y doce tablas correctamente creadas.

Todo el entorno ha sido migrado al servidor UOC (fp064.techlab.uoc.edu) mediante SCP/SFTP sobre SSH (puerto 55000), con respuesta HTTP/1.1 200 OK confirmada como evidencia final.
